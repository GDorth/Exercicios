HTML 5 - O esqueleto

HTML 5 é a linguagem de marcação padrão usada para criar páginas da web. É a espinha dorsal de praticamente todas as páginas da web e desempenha um papel fundamental na estruturação e apresentação de conteúdo na internet.

A principal função do HTML é definir a estrutura lógica e semântica de uma página da web, ou seja, ele define os elementos e sua organização hierárquica. Ele usa tags (etiquetas) para envolver o conteúdo e atributos para fornecer informações adicionais sobre esses elementos. Os navegadores lêem interpretam essas tags e exibem o conteúdo de acordo com suas especificações.

A tag âncora é usada para inserir links.

Incluindo Listas

Existem 3 tipos de listas:

Lista Ordenada
Usadas quando desejamos listar dados com a necessidade de representar a sua ordenação de forma numérica ou alfabética.
Lista Não Ordenada
Usadas quando não há necessidade de listar ordenadamente.
Lista de Definição
Usadas quando precisamos listar itens e atribuirmos uma descrição a eles.

CSS 3 - A skin

O CSS 3 é a versão mais recente do CSS, que é uma linguagem de estilo utilizada para definir a apresentação visual de documentos escritos em linguagens de marcação, como HTML. O CSS3 introduziu várias melhorias e recursos novos em relação às versões anteriores do CSS.

Tanto a HTML quanto a CSS são linguagens interpretadas diretamente pelo browser e que não precisam ser compiladas – exceto a CSS quando se utiliza pré-processadores.
